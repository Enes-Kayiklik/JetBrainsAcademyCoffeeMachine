Grades Open on JetBrains Academy

Find the number of "Ds", "Cs", "Bs" and "As" for the last test on informatics in the class consisting of n students. The program gets number n as input and then gets the grades themselves (one by one). The program is to output four numbers in a single line which is the number of "D", the number of "C", the number of "B" and the number of "A" grades.

Numbers represent grades this way: 2 is "D", 3 is "C", 4 is "B", and 5 is "A".


Sample Input:
12
3
5
5
5
4
5
3
4
5
3
3
2

Sample Output:
1 4 2 5


Memory limit: 256 MB
Time limit: 5 seconds
