Printing each word in a new line Open on JetBrains Academy

Write a program that reads five words from the standard input and outputs each word in a new line. First, you need to print all the words from the first line, then from the second (from left to right).

 

The words can be placed in the input stream in any way.

 

To solve the problem, use Scanner .


Sample Input:
This course
teach you Kotlin
Sample Output:
This
course
teach
you
Kotlin

Sample Input:
one two three four five
Sample Output:
one
two
three
four
five


Memory limit: 256 MB
Time limit: 8 seconds
