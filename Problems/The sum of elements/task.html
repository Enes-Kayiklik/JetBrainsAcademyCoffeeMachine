The sum of elements Open on JetBrains Academy

Find the sum of all the elements of a sequence, ending with the number 0.

The number 0 itself is not included in the sequence and serves as a sign that the sequence ended. Although if you add 0, nothing bad will happen :)


Sample Input:
3
6
8
0
Sample Output:
17


Memory limit: 256 MB
Time limit: 1 second
