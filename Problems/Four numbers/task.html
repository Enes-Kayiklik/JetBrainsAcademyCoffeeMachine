Four numbers Open on JetBrains Academy

Write a program that reads four integer numbers from one line and prints them each in a new line. In the input, numbers are separated by one or more spaces.

Use nextInt()  to read numbers ignoring extra spaces.


Sample Input:
101 102 103 104
Sample Output:
101
102
103
104


Memory limit: 256 MB
Time limit: 8 seconds
